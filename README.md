# ☕ Java

자바 학습을 위한 리포지터리입니다.  
Java 문법부터 Servlet, JSP, JDBC 등을 실습하며 기초 백엔드 역량을 다지는 것을 목표로 합니다.

---

## 📌 공통 학습 목표

- 자바 문법 및 객체지향 개념 실습
- HTTP 요청 흐름 및 서블릿 동작 방식 이해
- JSP와 서블릿을 활용한 MVC 패턴 구현
- JDBC를 이용한 데이터베이스 연동
- 톰캣 서버 배포 및 웹 환경 설정 경험

---

## ⚙️ 개발 환경

- Java 11  
- Apache Tomcat 10.1  
- JDBC / DBCP  
- JSP / Servlet  
- OracleDB or H2 DB
