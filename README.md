# ☕ Java Web Study

자바 웹 개발 학습을 위한 리포지터리입니다. CRUD 구현, 서블릿(Servlet), JSP 등을 활용한 기초적인 웹 프로그래밍을 실습하며 백엔드 개발 역량을 키우기 위한 목적입니다.

---

## 📌 학습 목표

- 자바로 CRUD 기능 직접 구현
- HTTP 요청 흐름 및 서블릿 동작 방식 이해
- JSP와 서블릿을 이용한 MVC 패턴의 기초 실습
- 톰캣 서버 배포 및 웹 환경 설정 경험

---

## ⚙️ 버전

- Java 11
- Apache Tomcat 10.1
- JDBC / DBCP
- JSP / Servlet
- OracleDB
